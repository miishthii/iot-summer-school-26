# 📖 Overview

This repository contains the solutions for **Module 4 and Module 5** of the IoT Summer School Assignment. The assignment covers IoT communication protocols, wireless networking, cloud connectivity, MQTT, HTTP, security concepts, LPWAN technologies, and multiple ESP32-based IoT application projects developed using **Wokwi**.

The projects demonstrate sensor interfacing, embedded programming, cloud communication, automation, environmental monitoring, and smart city applications using the ESP32 platform.

---

# 🎯 Objectives

- Learn IoT communication protocols (MQTT, HTTP)
- Understand IoT security concepts
- Explore LPWAN technologies such as LoRa
- Design and implement ESP32-based IoT systems
- Simulate embedded applications using Wokwi
- Integrate sensors, cloud platforms, and dashboards
- Develop practical smart city and automation solutions
