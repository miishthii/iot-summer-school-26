# IoT Week 1

## Overview

This folder contains the Week 1 assignments completed during the **IoT Summer School 2026**. The exercises focus on learning Arduino programming fundamentals, digital input/output, arrays, LED control, push buttons, and basic embedded system concepts.

---

## Projects Included

### LED Blink
https://www.tinkercad.com/things/kqjr9hdmkIG-ledblinker10?sharecode=rWVNtNH202q9rJTsXMAxGptGkSNriEg-DsWpZ1nM4-k

* Basic Arduino program to blink an LED.
* Demonstrates digital output and timing using `delay()`.

### Pushbutton LED
https://www.tinkercad.com/things/kGuYhXHwCc2-push-button-led?sharecode=GxAAzYkpTdv7cQGF8fa3RQqk74Y2FjKtG_w3ySCox3E

* Controls an LED using a push button.
* Demonstrates digital input and button state detection.

### LED Fade Array
https://www.tinkercad.com/things/bykP8v0A084-led-fade-array?sharecode=PjNu2ATi2N720VUWKqGk_50AviI8xY_MZN_FXvNw2LU

* Fades multiple LEDs using PWM and arrays.
* Demonstrates analog output and array-based programming.

### LED 4×4 Matrix
https://www.tinkercad.com/things/2A2ARfAASSv-4x4-using-matrix?sharecode=dXuMuSWGff-Zx0TzCKy5rMadOiUBljEnZGCHO853vW8

* Controls a 4×4 LED matrix using row and column multiplexing.
* Demonstrates matrix scanning and efficient pin usage.

### LED 4×4 Array
https://www.tinkercad.com/things/5st4dv4BNnc-4x4-using-array?sharecode=a3iKWJEVH7OaRgw92lsMvMeYP2-Y_jEjAHhLKItXGoE

* Demonstrates structured programming with arrays.

### Seven Segment Dice
https://www.tinkercad.com/things/eucAGBSPh9e-dice?sharecode=ILMGCMJuoNSPPP9CUc3OQnMpfqskWZUXv2CM0oS7WzI

* Simulates a digital dice using a seven-segment display.
* Displays random numbers from 1 to 6.

### Reaction Time Game
https://www.tinkercad.com/things/8gW5UgFPHc4-reaction-time-game?sharecode=7Jq4pOxRMI3OKRdamhR0Iu2RkNtMp0n6T_zQDVU-DAc
